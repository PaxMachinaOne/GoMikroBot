# GoMikroBot Community

Welcome to the GoMikroBot community!

## Philosophy
We believe in **Small Core, Big Capabilities**.
- **Agents as Binaries**: Easy to deploy, easy to run.
- **Text-First**: Unix philosophy. Input text, output text.
- **Composable**: Tools are interfaces.

## Channels
- **GitHub**: [Discussions](https://github.com/kamir/gomicrobot/discussions)
- **Discord**: [Join Server](https://discord.gg/placeholder)

## How to Contribute
1.  **Core**: Go code in `internal/`.
2.  **Tools**: Implement the `Tool` interface.
3.  **Docs**: Improvements to `docs/`.

## Governance
This project follows a benevolent dictator model currently led by @kamir, with open contributions welcome.
