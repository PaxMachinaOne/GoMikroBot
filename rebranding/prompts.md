# Branding Prompts for NanoBanana

Use these prompts to generate visual assets for GoMikroBot.

## 1. Main Logo (Icon)
**Prompt:**
> A minimalist, modern app icon for "GoMikroBot". The design should feature a cute, tiny robot with a metallic finish, integrating elements of the Go programming language mascot (gopher) or color palette (cyan/blue). The robot should look efficient and speedy. Vector art style, flat design with subtle gradients, white background.

## 2. CLI Header / Banner (ASCII Art style but visual)
**Prompt:**
> A wide banner image for a GitHub README. Dark mode background (deep slate/black). Text "GoMikroBot" in futuristic, monospaced font glowing in cyan. On the left, a small 3D render of the robot mascot assembling code blocks. Tech grid lines in the background. Cyberpunk aesthetic but clean.

## 3. Social Media Cover
**Prompt:**
> A social media cover image. Abstract visualization of a "swarm" of tiny, helpful microbots working together to build a structure. Colors: Cyan, Electric Blue, Silver. The vibe should be "Cooperation, Speed, Precision".

## 4. "Soul" Visualization (for documentation)
**Prompt:**
> An artistic representation of an AI's "soul" or "core memory". A glowing, crystalline orb containing floating lines of code and geometric shapes, suspended in a high-tech containment unit. Soft, ethereal lighting.
